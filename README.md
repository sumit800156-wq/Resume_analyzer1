# AI Resume Analyzer

AI Resume Analyzer is a Streamlit-based web application that analyzes resumes and provides an ATS (Applicant Tracking System) score.

## Features

- Upload Resume (PDF)
- ATS Score Calculation
- Resume Content Analysis
- Skill Extraction
- Keyword Matching
- Resume Improvement Suggestions
- Attractive User Interface

## Technologies Used

- Python
- Streamlit
- PyPDF2
- NLTK
- Scikit-learn
